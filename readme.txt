Steps for running the project:

Step 1: First Open Visual Studio Code IDE
Step 2: Then Open the project
Step 3: Open the HTML file in a web browser:
      •	Right-click on the file and choose "Open with" from the context menu.
      •	Select your preferred web browser from the list of available programs.
      •	The web browser will open and load the HTML file.
Step 4: Interact with the virtual assistant:
      •	Once the web page is loaded in the browser, you can start interacting with your virtual assistant.
      •	Allow microphone to access your input.
      • Click buttons, or perform any required actions to interact with the virtual assistant.
Step 5: Observe the responses:
      •	As you interact with the virtual assistant, it will process your input and generate appropriate responses.
